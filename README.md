# Ещё успеваешь 🤓

Это агрегатор анонсов российских научных конференций - живет на этом [сайте](https://conferences-hub.com/), - на которые открыт прием заявок. Вот как это работает по шагам:

**Шаг 1** (*crawl*). Специальный краулер ("Падальщик") собирает новости и анонсы с сайтов российских вузов. Их полный списко см. ниже. 

**Шаг 2** (*classify*). Среди новостей отбираются только те, которые содержат объявления о будущих конференциях - решается задача классификации. Это делается через: (1) первичный фильтр по ключевым словам; и далее (2) через (мажоритарное) голосование трех open-source LLM: [Qwen3-8B](https://ollama.com/library/qwen3:8b), [Gemma 3 12B](https://ollama.com/library/gemma3:12b) и [Mistral NeMo 12B](https://ollama.com/library/mistral-nemo:12b). В будущем я планирую перейти на более простой классификатор, когда llms дадут достаточно обучающего материала.

**Шаг 3** (*extract*). Среди отобранных записей извлекаются ключевые поля: даты проведения, дедлайн, научное направление на основе рубрикатора OECD, немного переосмысленного Вышкой (см. [тут](https://www.hse.ru/science/scifund/klass_new)) и прочее. Это делается также через ансамбль и мажоритарное голосование. Если по полям с датами проведения или дедлайнами сложилась ситуцаия голосования 1:1:1, то такая запись уходит на ручную проверку. В случае такого же исхода голосования по другим полям (например, формат проведения) - записи ставится пометка (флажок ⚑ на сайте), но она публикуется. 

Далее в разделе **Сводки** дана информация по "здоровью" обновления данных и источникам.
